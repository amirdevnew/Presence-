# Presence +
Presence+ is a web-based attendance management system built with QR Code technology to streamline attendance tracking for employees and students. The system enables users to record attendance efficiently through QR code scanning while improving accuracy and reducing manual processes.
In this project, I worked independently and was responsible for the complete development of the application on the Web platform. The application was built using Java for backend development, HTML for structuring the application, and CSS for designing an attractive and user-friendly interface.
